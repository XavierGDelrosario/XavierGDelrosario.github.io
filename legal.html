Legal Mediation Database

A sample of the normalized mediation database, loaded in memory and queried right in your browser.

Manage a client

Add a new client, or pick an existing one to edit their contact information or their mediation relationships with other clients — the same insert / update / delete operations the PHP admin pages perform, kept consistent across the linked tables.

Query the database

Choose a query — each runs the same relational logic as the PHP application: selection, projection, joins, aggregation, and a division query. Clients you add or remove above appear here too.
